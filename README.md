### Основное

- Java 17
- Spring Boot 3.1.5
- Springdoc 2.2.0

### Запуск:

Из корня проекта:

```
mvn clean package
docker build ./ -t <img_name>
docker run --name <cont_name> -p 8080:8080 <img_name>
```

### Описание API:

- OpenAPI находится на эндпоинте /swagger-ui/index.html

Примеры входных и выходных параментров:

##### 1. Стандартный пример

Входные данные:
```
"aaaaabcccc"
```

Выходные данные:
```
{
  "a": 5,
  "c": 4,
  "b": 1
}
```

##### 2. Пустая строка

Входные данные:
```
""
```

Выходные данные:
```
{}
```
